Detailed Description

Time-extrapolation strategies for Black volatility term structures.

This class provides static methods to extrapolate variance beyond the maximum time of a BlackVolTermStructure, supporting both surface (strike-dependent) and curve (ATM) cases.

Different extrapolation strategies are available via the Type enum.
